# dust-attack

**@dust-attack** is a slack bot in overflow-zone

![example](https://user-images.githubusercontent.com/11442383/28674340-15d68634-7320-11e7-95d2-8dfc559b6cd4.png)

## How to use

1. You must modify `conf.py`.
2. Run below command.

```bash
python bot.py
```
